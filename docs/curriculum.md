# カリキュラム全体像

## 研修の目的

本研修プログラムは、法務担当者がAIツールを効果的に活用し、業務効率化と品質向上を実現するためのスキルを段階的に習得することを目的としています。

## 研修の構成

### 第1段階：AI基礎研修（初心者向け）

**目標**: AIとLLMの基本概念を理解し、基本的な操作方法を習得する

- AI・LLMの基本概念と仕組み
- 主要なAIサービスの概要と特徴
- プロンプトエンジニアリングの基礎
- 法務分野でのAI活用事例
- AIの倫理的・法的課題

**所要時間**: 約6時間（講義4時間、演習2時間）

### 第2段階：LLMの基礎と実践研修

**目標**: 法務業務に即したLLM活用テクニックを習得する

- LLMの強みと弱みの理解
- 高度なプロンプト設計手法
- 法務文書の要約・分析・生成
- 効率的なAI活用のためのチェックリスト
- 実務での活用シナリオと注意点

**所要時間**: 約8時間（講義4時間、演習4時間）

### 第3段階：API活用とローカル環境での実践研修

**目標**: APIを活用した基本的なアプリケーション開発スキルを習得する

- API連携の基礎知識
- OpenAI APIとGemini APIの活用方法
- Python基礎とAI連携
- データベース（Supabase）の基本
- 簡易アプリケーション設計と実装

**所要時間**: 約12時間（講義6時間、演習6時間）

### 第4段階：高度なAI活用とアプリデプロイ研修

**目標**: 高度なAI活用技術とアプリケーションのデプロイ方法を習得する

- AIコーディング支援ツールの活用
- Docker環境の構築と活用
- RAG（検索拡張生成）の実装
- ファインチューニングの基礎
- アプリケーションのデプロイと運用

**所要時間**: 約14時間（講義6時間、演習8時間）

## 修了要件

- 各段階の演習課題をすべて完了すること
- 最終プロジェクトとして、法務業務に役立つAIアプリケーションを開発し発表すること
